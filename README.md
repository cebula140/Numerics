# Numerics

A lightweight numerical library in C++ 23 made for learning. Designed to be simple and readable.

## Algorithms

### Differentiation
- Central derivative (2-point, O(h²))
- Four-point derivative (O(h⁴))

### Integration
- Composite Simpson's rule
- Adaptive Simpson's rule

### Root finding
- Newton's method
- Bisection method

## Build

```
cmake -B build
```
